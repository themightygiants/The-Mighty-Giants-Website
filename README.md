# The Mighty Giants Site

Modern responsive brand website for The Mighty Giants using Next.js App Router, React, Tailwind CSS, and TypeScript.

## Run locally

1. Install Node.js 20+ from [https://nodejs.org](https://nodejs.org)
2. In this folder, install packages:

   ```bash
   npm install
   ```

3. Start dev server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` Home
- `/about` About
- `/merch` Merch
- `/booking` Booking
- `/media` Media

## Asset mapping

- Backgrounds in `public/images/backgrounds`
- Logos in `public/images/logos`
- Headshots in `public/images/headshots`
